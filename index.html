NaowhUI Token Generator

Generates a free token to unlock NaowhUI. Disclaimer: according to World of Warcraft Add-On development rules, addons must be free of charge and have fully visible source code. See the official UI Add-On Development Policy.

Debug values

Optional. If empty, server time is inferred from epoch. You can run /run local x=GetServerTime() print(x) in game to get the server timestamp.
